# Entities, Properties and Behaviours:

## Examples:

### 1. Bird:

| Properties | Behaviours |
| --- | --- |
| Color | Fly |
| Weight | Eat |
| Height | Sleep |
| Wingspan | Lay Eggs |
| Beak Length | Sing |
| Species | Migrate |

### 2. Car:

| Properties | Behaviours |
| --- | --- |
| Make | Start |
| Model | Stop |
| Year | Accelerate |
| Color | Brake |
| Engine Size | Turn |
| Fuel Type | Park |
